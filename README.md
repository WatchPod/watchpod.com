# WatchPod

The WatchPod website, built with Next.js, React, and Tailwind CSS.

## Setup

```bash
npm install
cp .env.example .env.local
```

## Environment Variables

| Variable | Description |
|---|---|
| `NEXT_PUBLIC_GOOGLE_SCRIPT_URL` | URL of the Google Apps Script that handles waitlist submissions |

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run start
```
